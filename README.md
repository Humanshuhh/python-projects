# 🐢 Turtle Race Game

A fun Python Turtle Graphics game where multiple turtles race to the finish line, and the player can bet on which turtle will win.

## 🎮 Features

- 6 colorful racing turtles
- User betting system
- Randomized race outcomes
- Graphical race track
- Winner announcement
- Simple and beginner-friendly code

## 📸 Preview

The game displays a race track with six turtles. Players choose a turtle before the race starts and watch the race unfold in real time.

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics
- Random Module

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/turtle-race-game.git
```

### 2. Navigate to the Project Folder

```bash
cd turtle-race-game
```

### 3. Run the Program

```bash
python turtle_race.py
```

## 🎯 How to Play

1. Run the program.
2. Enter the name of the turtle you want to bet on.
3. Click anywhere on the screen to start the race.
4. Watch the turtles race to the finish line.
5. See if your prediction was correct!
